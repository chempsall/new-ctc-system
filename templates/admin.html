Admin

All actions require the admin token, entered once below and reused for every request on this page.

Import staff list

Re-reads the staff list Excel file and updates the database. Rebuilds the summary cache.

Import PAR actuals

Re-reads the PAR actuals export and updates project metadata and inception-to-date figures.

Rebuild summary cache

Forces a full rebuild of the dashboard summary JSON. Runs automatically after imports.

Clean up deleted CTC files

Checks every RTC on record and archives any that have no future allocations and haven't been opened in 30+ days. Runs automatically every night — use this to force it immediately.

Delete RTC

Permanently deletes an RTC and all its allocations. Load the list first, then select one to delete.

Import log

Recent import runs with row counts and any errors.

Configuration

Current server configuration (non-sensitive settings only).
