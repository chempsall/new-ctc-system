Actions

Import staff list
Re-reads the staff list Excel file and updates the database. Rebuilds the summary cache automatically on completion.
Re-link pending RTCs
Checks all unlinked RTCs against current PAR data and links any where the project number and task order now match a real Horizon record. Runs automatically after each nightly PAR import.
Refresh linked RTCs from PAR
Re-pulls department, project name, task name, customer, PD and PM from the PAR record for all currently linked RTCs. Use this after a PAR import to ensure all linked RTCs have the latest information.
Import PAR actuals
Re-reads the PAR actuals export and updates all project metadata. Rebuilds the summary cache automatically on completion.
Process staff leavers
Transfers future allocations from staff who have left to the grade-equivalent generic. Zeros out leaver rows on special RTCs without replacement. Runs automatically every night.
Initialise / refresh special RTCs
Creates or updates the Annual Leave & Public Holidays, Training, and Day Release & Study Leave RTCs for all departments. Adds new staff, pre-fills bank holidays, and removes allocation rows older than 12 months. Runs automatically every night.
Rebuild summary cache
Forces a full rebuild of the dashboard summary JSON. Runs automatically after every import — use this if the dashboard appears out of date.
Archive old RTCs
Archives RTCs that have no future allocations and haven't been opened in 30+ days. Archived RTCs are hidden from the default list but never deleted. Runs automatically every night.
Import log
Shows the 10 most recent import runs with row counts and errors.
Configuration
Current server configuration — non-sensitive settings only. Useful for verifying file paths and scheduler times.
