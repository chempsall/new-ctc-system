Resource Forecast — Admin

All actions require the admin token in the Authorization header. Use the API directly or via the buttons below.

Import staff list

Re-reads the staff list Excel file and updates the database. Rebuilds summary cache.

Import Horizon projects

Re-reads the Horizon export and updates project metadata. Does not affect allocations.

Import PAR actuals

Re-reads the PAR actuals export and updates inception-to-date figures.

Rebuild summary cache

Forces a full rebuild of the dashboard summary JSON. Runs automatically after imports.

Clean up deleted CTC files

Checks every CTC file on record against disk and removes any whose file no longer exists, along with its allocations. Runs automatically every night — use this to force it immediately.

Conflict flags

Projects flagged due to duplicate filenames with placeholder project numbers.

Start date changes

Projects where the CTC start date was changed after initial setup.

Import log

Recent import runs with row counts and any errors.

Configuration

Current server configuration (non-sensitive settings only).
